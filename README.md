# Collabs - Aplikasi Kolaborasi Tugas Tim

Collabs adalah aplikasi web untuk manajemen tugas dan kolaborasi tim kecil (3-4 orang). Dibangun dengan PHP Native, MySQL, dan jQuery.

## Fitur Utama
- ✅ Manajemen User (Admin & Member)
- ✅ CRUD Task & Subtask
- ✅ Status: Proses, Selesai, Evaluasi (auto-evaluate deadline)
- ✅ Prioritas: High, Medium, Low
- ✅ Deadline dengan visual warning
- ✅ Komentar dengan link attachment
- ✅ In-App Notifications (real-time dengan polling)
- ✅ Activity Log sederhana
- ✅ Drag & Drop custom urutan subtask
- ✅ Filter & Pencarian
- ✅ Arsip Tugas (soft delete manual)

## Teknologi
- PHP Native (tanpa framework)
- MySQL
- jQuery/AJAX
- Bootstrap 5
- Laragon (environment development)

## Instalasi
1. Clone repository
2. Import database dari file `database/collabs.sql`
3. Konfigurasi koneksi database di `config/database.php`
4. Jalankan di localhost
